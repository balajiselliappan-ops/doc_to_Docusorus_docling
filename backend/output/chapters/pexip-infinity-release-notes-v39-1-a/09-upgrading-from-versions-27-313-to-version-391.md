## Upgrading from versions 27-31.3 to version 39.1

If you are running a software version between v27 and v31.3 inclusive, you must first upgrade to version 31.4, then upgrade again to version 35, and then upgrade again to version 39.1.

![Image](/img/docs/pexip-infinity-release-notes-v39-1-a/image_000005_465f55b4047e54c3ab0a3fd95331246eb0ce1b246a84e2694ac95f59641d1a28.png)

All previous versions listed above are available to download from the Pexip website.
