## Upgrading to version 39.1
- If you are running a software version between v31.4 and v34.2 inclusive, you must first upgrade to version 35, and then upgrade again to version 39.1; see [Upgrading from versions 31.4-34.2 to version 39.1](#upgrading-from-versions-314-342-to-version-391).
