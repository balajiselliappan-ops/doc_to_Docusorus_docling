## Upgrading from versions 22-26 to version 39.1

If you are running a software version between v22 and v26 inclusive, you must first upgrade to version 27.4, and then upgrade again to version 31.4, and then upgrade again to version 35, and then upgrade again to 39.1.

- All previous versions mentioned above are available to download from the Pexip website.
