## Pexip Infinity
|   Ref # | Resolution                                                                                                                                                 |
|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
|   47585 | Resolved an issue in v39.0 in deployments with multiple custom themes that could cause the wrong watermark to be displayed.                                |
|   47170 | Resolved an issue where classification indicators could not be set via the Client API with a call using a Proxying Edge Node.                              |
|   44858 | Resolved an issue in a gateway call using an Adaptive Composition or Teams-like layout, where content classifications and banner messages were not shown.  |
|   44427 | Resolved an issue that in some cases caused multiple participants to be created when a call was de-escalated from being transcoded to a direct media call. |
